📌 Project Overview

This project analyzes Human Resources (HR) data to uncover patterns related to:

Employee attrition
Department-wise performance
Salary distribution
Employee satisfaction
Experience and promotion trends
Workforce demographics

The goal is to help organizations make data-driven HR decisions and improve employee retention and productivity.

📊 Features
Data cleaning and preprocessing
Exploratory Data Analysis (EDA)
Interactive visualizations
Attrition analysis
Department and gender distribution insights
Salary and performance correlation
Dashboard/report generation
🛠️ Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Plotly
Jupyter Notebook / VS Code
